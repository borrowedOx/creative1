Step 1

Access your AWS hosting account.

Assuming you followed along with the instructional guide on learning suite, you should have a .pem file somewhere on your computer. If you're on a Mac or Linux computer you might get tired of typing out an entire line every time you want to log in. Well, there's a fix for that.

You can add what's called an alias to your terminal. It's a way to make shortcuts for commands. Depending on what system you're on you'll need to create a file called either .bashrc, .bash_profile, or .bash_aliases (try vim .bash_profile first). In that file you'll add something like this: alias cs201r="ssh -i /PATH/TO/YOUR/.PEM/FILE.pem ubuntu@[YOUR AWS INSTANCE NAME]"

Close out your current shell and the next time you want to log in just typue cs201r (or whatever you want your command to be
